hexo添加rss和站内搜索

发表于 2015-09-24

添加RSS

首先这个是在next主题下的设置方法。

编辑主题config:theme/next/_config.yml

  1. 安装rss插件 npm install hexo-generator-feed
  2. rss: false，这将会禁用Feed链接。
  3. rss:，当值为空的时候，默认会使用站点的 Feed 链接。在此之前需要使用 hexo-generator-feed 插件生成 Feed。

编辑站点config:_config.yml 添加

添加站内搜索 tinysou

选择tinysou的原因：免费，对中文支持好，next支持。

  1. 注册tinysou，可以直接用github登陆
    创建新的engines

  2. 在engine的爬虫页面中添加自己站点的域名

  3. 在主站点config:_config.yml添加tinysou_Key: yourtinysou_engineKey
