mac下用github+hexo搭建个人博客

发表于 2015-09-23

安装hexo

首先需要安装nodejs, git
然后安装hexo

安装hexo会遇到npm链接不上的情况，这时可以修改node的repository。编辑~/.npmrc

可以选择安装hexo的插件

如果是迁移原来的hexo，需要删除根目录下的.deploy_git

hexo 文件夹init

新建一个文件夹hexo并进入

该文件夹下会产生如下目录

  • _config_yml // 配置文件
  • db.json // 数据
  • debug.log // 日志
  • _node_mudules // nodejs 相关依赖
  • package.json // 配置依赖
  • scaffolds // 工具模板
  • source // blog正文
  • themes // 皮肤

创建github page

登陆自己的github并创建New repository，注意name必须和用户名一致，如thousandhu.github.io，等待一段时间后github page会自动创建成功

第一篇blog

运行hexo new hello会在/hexo/source/_posts/下生成hello.md，之后修改hello.md即可。支持markdown语法~

更换主题

我用的是yilianext
将主题文件下载到theme中。
修改_config.yml : theme: yilia
修改其他配置，主要是使用了多说作为评论插件，创建一个多说工程，将工程名填入主题的_config.yml即可
